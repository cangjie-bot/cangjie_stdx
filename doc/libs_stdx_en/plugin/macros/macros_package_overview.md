# stdx.plugin.macros

## Overview

This package provides users with an [@astPlugin](./macros_package_api/macros_package_macros.md#astplugin-macro) macro for annotating AST plugin classes and automatically generating plugin registration code.

## API List

### Macros

|              Macro          |           Description           |
| --------------------------- | ------------------------------- |
| [astPlugin](./macros_package_api/macros_package_macros.md#astplugin-macro)     | Annotates an AST plugin class and automatically generates plugin registration code. |
