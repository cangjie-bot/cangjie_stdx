# stdx.plugin.macros

## 功能介绍

该包为用户提供了一个 [@astPlugin](./macros_package_api/macros_package_macros.md#astplugin-宏) 宏，用于标注 AST 插件类并自动生成插件注册代码。

## API 列表

### 宏

|              宏          |           功能           |
| --------------------------- | ------------------------ |
| [astPlugin](./macros_package_api/macros_package_macros.md#astplugin-宏)     | 用于标注 AST 插件类，自动生成插件注册代码。 |
